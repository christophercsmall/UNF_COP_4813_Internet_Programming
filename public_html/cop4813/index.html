Assignment 1

Assignment Description:

Static web pages created using HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) remain an essential part of web design. This first internet programming assignment addresses fundamental concepts and practices of web development and design while focusing the importance of structure and standards used in development. In order to understand how static web pages are created and designed, this assignment requires the student learn how to read and write HTML and CSS code, link external CSS files to HTML files, and basic Linux commands to create, read, and manipulate files from the terminal. Also important, is how to create folder structures that keep a web site's HTML and CSS files organized.



Reflection About What I Learned:

This assignment helped me learn about how much can be accomplished without the use of GUIs (Graphical User Interface). I have not used linux terminal commands in a long time, so now I have a greater understanding now of the value of commonly used commands in the CLI (Command Line Interface) and the difference between and proper use of relative and absolute paths when linking web pages and navigating the linux CLI. While absolute paths can always be used to link pages within the same site, it is bad practice because a domain name change would require all absolute paths be changed in the entire project.

Learning Objectives for this Assignment

  • Fundamental Development of Static Web Pages Using HTML
  • Using Linked, Inline, and External CSS (Cascading Style Sheets) to Arrange and Format HTML Content
  • Using Basic Linux Terminal Commands to Create, Read, and Concatenate Files
  • Creating Folder Structures for Organizing Web Page Files
  • General Web Page Design and Aesthetics

Assignment 4

Assignment Description:

This assignment introduces HTML5 Canvas as a solution for creating styled text and animations using only HTML and JavaScript. Creating a canvas means designating an area within a web page for displaying colors and shapes that appear static or dynamic depending on the drawing logic designed by the programmer. Shapes and colors are drawn and/or erased in accordance with triggered events such as time intervals or mouse clicks. By drawing, clearing, and redrawing to slightly different dimensions, an appearance of animation can be achieved. There is a deep emphasis on x and y coordinates used as reference points each time the canvas is updated. This assignment also extends the power and utility of JavaScript.



Reflection About What I Learned:

Though I enjoy programming of any sort, this assignment was the most daunting for me. Using JavaScript and HTML for animation takes a step outside of engineering and focuses primarily on aesthetic. I can see how including a well made canvas has the potential to enhance an experience on the web. I have dabbled a bit in using sprite sheets to make retro-style RPGs, so I could get a nice refresh on using frame rates and simulating collision bounderies.

Learning Objectives for this Assignment

  • Using JavaScript and HTML5 to Create Animations within a Web Page
  • Using and Updating Arrays and Objects
  • Canvas Manipulation using Pure JavaScript


Assignment 5

Assignment Description:

This assignment introduces PHP (PHP: HyperText Preprocessor) as a way to generate HTML and logic output solely from the server as opposed to doing so on the client like with JavaScript. Session variables are introduced as a way to create persistent data which is commonly used to maintain user login status across many pages as the user navigates through the site. With this server-side scripting language, the user is oblivious to the processes that go on behind the scenes. This is of fundamental value to authentication methods and other processes that the user should not be allowed to see or modify.



Reflection About What I Learned:

This was my first time ever coding with PHP. It was a bit uncomfortable at first because server-side scripting was a concept I had not had enough experience with in order to find any value that could be applied to anything I had worked on in the past few years of studies in programming. I now have a better understanding of why some processes absolutely need to be completed at the server before the user sees a single line of HTML rendered on their screen. I used to think I would rarely have a need for PHP, but now I can see how it could have solved so many problems I used to work around and how PHP could be one of my most used tools in future projects.

Learning Objectives for this Assignment

  • Learn How to Manage Login Requests Processed on the Server
  • Use Session Variables to Create Persistent Data Across Pages
  • Learn How to Open and Close Text Files to Read and Write
  • Fetch Financial Data from Sources on the Internet
  • Create a Single Project Combining Multiple Types of Code, Including HTML, CSS, JavaScript, and PHP.


Assignment 6

ERD

Assignment Description:

This assignment expands upon our knowledge of PHP and JavaScript while introducing MySQL as a database solution for web applications. Using PHP, queries can be issued to a database in order to create tables and modify records that exist within. Using DML (Data Manipulation Language), it is possible to manipulate relations with SQL keywords like SELECT, INSERT, UPDATE, and DELETE. This assignment also stresses the importance of referential integrity within a database and basic understanding of how databases can be compromised when users are allowed to submit unsanitized strings of text through inputs in forms containing text boxes and various other types of input methods.



Reflection About What I Learned:

This module was not particularily tough or time consuming in itself, but alongside every other project due around this same week it became a chore. Continuing to use PHP remains a rewarding experience to me; I just wish I had more time to appreciate it without worrying about other deadlines.

Learning Objectives for this Assignment

  • Learn How to Connect to and Use a Remote Instance of a MySQL Database
  • Use PHP to issue SQL Commands to Select, Insert, Update, and Delete Records in a Table
  • Continued Honing of JavaScript Skills
  • Thwart Possible SQL Injection Attempts
